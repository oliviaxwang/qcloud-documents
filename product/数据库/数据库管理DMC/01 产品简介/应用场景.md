
## 场景一：统一的数据库管理平台 
DMC 提供统一的数据库管理平台，支持多种数据库接入，界面操作可视化，方便用户进行数据库表结构、视图等操作变更管理。

## 场景二：SQL 审批 
- DMC 提供 SQL 安全审计，用户提交 SQL 数据变更后，对 SQL 语法进行安全审计，保证 SQL 输入的规范性和安全性。
- 对于企业用户，通过工单管理，规范 SQL 变更、权限申请等流程，提升企业管理数据库的效率。
